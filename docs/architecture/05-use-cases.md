# 05 — Casos de Uso

## 1. Objetivo do Documento

Este documento descreve os principais casos de uso do projeto Forum, servindo como ponte entre a descoberta do domínio, a linguagem ubíqua, o event storming e a futura implementação da aplicação.

## 2. Escopo

O Forum é uma plataforma de comunidades híbrida, com leitura parcialmente pública e interação restrita a usuários autenticados e autorizados.

## 3. Atores

- Visitante
- Membro
- Moderador
- Administrador

## 4. Módulos Funcionais

- Comunidades
- Categorias
- Canais
- Tópicos
- Respostas
- Tags
- Soluções
- Moderação
- Anúncios
- Eventos
